website
This website is inspired from the Pantone color authority, though not in any way affiliated. I just love digital color.

color
Choosing a color scheme is a huge part of developing anything digital. From a simple user interface to complete branding marketing materials, color is a critical factor in how approachable a design is. I want to make an interface which feels fresh, and this is the beginning.

pantone
The No.1 cool thing about Pantone palettes has got to be the carefully thought out relationships between every range of colors. Beautiful complementary tones. My favourite palette on this website is the `Pantones of Fall 2016`, check it out by clicking `change palette` in the bottom left corner. Have a play around - you won't break anything. legibility
In recent years thin fonts have become a standard in quickly achieving a professional typeface. So what's good? Let's compare.

I'm bold, bold as love.

I'm normal, normal as love.

I'm light, light as love.

I'm thin, thin as love.

On this page we are using the light `Lato` font for the body text. Thin is a little too skinny to being truly legible for body text but with a large font size, say 40px, it looks good. Always a toss up between aesthetic and usability. Though are blessed because beauty is in the eye of the beholder. I reckon,

Thin fonts can pack a punch.

star wars vs good code
A crucial usability factor when using a code editor is the theme. My code editor is Star Wars themed - this only serving a purpose of enjoyment - and STILL I type more stuff, I'm more engaged. How come that's even a thing? Dunno why, but I sacrifice legibility for style. It's less readable than it is with a light theme yet I will have more concentration on it because of the visual appeal. Like the old geezer with the pink speedos on the beach all summer. Not really like that but we've all been there.
